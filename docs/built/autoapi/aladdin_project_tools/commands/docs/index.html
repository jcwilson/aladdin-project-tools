aladdin_project_tools.commands.docs¶

Commands for working with the project’s documentation.

Once you poetry install this project, you can invoke these commands at the command line.

When not in a poetry shell¶
$ poetry run docs --help
When in an activated poetry shell¶
$ docs --help

build(show: bool = typer.Option(False, help='Open the docs in a browser after they have been built.'), sphinx_args: List[str] = typer.Argument(None))[source]¶

Build the documentation.

Parameters

  • show – Open the docs in a browser after they have been built, defaults to False.

  • sphinx_args –

    Any remaining arguments will be passed to the underlying sphinx_main() function that actually builds the docs.

    Important

    If these arguments contain - or -- flags, you will need to preface this argument list with --.

Examples:

Simple build¶
$ docs build
Open the docs afterwards¶
$ docs build --show
Perform a full rebuild with debug logging¶
$ docs --log-level DEBUG build -- a
